# AICoder 使用说明书

[❓ 常见问题 (FAQ)](faq.md)

欢迎使用 **AICoder**！本工具专为简化多种 AI 编程命令行工具（Claude Code, Codex, Gemini CLI）的配置、多项目管理与启动流程而设计。

以下是详细的操作指南：

## 1. 启动与环境检测
首次运行程序时，AICoder 会自动检查您的系统环境：
*   **依赖检测**：检测 Node.js 及其他必要运行时。
*   **工具安装**：程序会自动检测并尝试安装/更新 `claude-code`, `codex`, `gemini-cli` 等工具到最新版本。
*   **启动窗口**：启动时会显示一个进度窗口，展示环境准备状态。

## 2. 侧边栏导航 (Sidebar)
AICoder 采用垂直侧边栏设计，您可以轻松切换不同的 AI 工具：
*   **Claude**：配置和启动 Anthropic Claude Code。
*   **Codex**：配置和启动 OpenAI Codex 相关的 CLI 工具。
*   **Gemini**：配置和启动 Google Gemini 相关的 CLI 工具。

## 3. 模型配置 (Model Settings)
在每个工具的面板中，您需要配置相应的 API Key。

1.  在侧边栏选择对应的 AI 工具。
2.  在主界面找到 **“模型设置”** 区域。
3.  **API Key**：在输入框中粘贴您的 API 密钥。如果配置成功，系统会记录并用于后续启动。
4.  **模型切换**：部分工具支持在不同的子模型或端点间切换。

## 4. 多项目管理 (Multi-Project)
您可以同时管理多个不同的代码项目，并为每个项目设置独立的路径和模式。

### 4.1 切换项目
*   在 **“Vibe Coding”** 区域可以看到项目选项卡（Tab）。
*   点击对应的项目名称即可快速切换。

### 4.2 项目管理
点击项目区域的 **“管理项目”** 按钮：
*   **添加项目**：添加新项目，设置独立的路径。
*   **重命名/删除**：管理现有的项目列表。

## 5. 设置项目参数
在选中某个项目后，可以对其进行具体设置：
1.  **项目目录**：点击 **“更改”** 重新选择代码所在的文件夹。
2.  **启动参数**：例如针对 Claude 的 **Yolo 模式**，勾选后会跳过权限询问。

## 6. 启动 AI 工具
1.  确保已选中有效的**模型**和**项目**。
2.  点击 **“启动”** 或 **“Launch”** 按钮。
3.  成功启动后，将弹出终端窗口（CMD/Terminal）进入交互界面。

## 7. 其他功能
*   **状态栏**：界面底部显示实时操作反馈和错误信息。
*   **多语言切换**：在标题栏或设置中切换界面语言。
*   **检查更新**：获取 AICoder 的最新版本。
*   **系统托盘**：右键点击托盘图标可快速启动工具或切换配置。
