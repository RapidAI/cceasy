# Claude Code Easy Suite 使用说明书

欢迎使用 **Claude Code Easy Suite**！本工具专为简化 Anthropic `claude-code` 命令行工具的配置、模型切换与启动流程而设计。

以下是详细的操作指南：

## 1. 启动与环境检测
首次运行程序时，工具会自动检查您的系统环境：
*   **Node.js**：如果未安装，程序会尝试自动通过 Winget 安装。
*   **Claude Code**：程序会自动安装或更新 `@anthropic-ai/claude-code` 到最新版本。
*   **注意**：如果进行了自动安装，程序可能会自动重启以应用更改。

## 2. 配置模型参数 (Model Settings)
在使用之前，您需要配置至少一个大模型服务商的 API Key。

1.  定位到主界面下方的 **"Model Settings" (模型设置)** 区域。
2.  点击选项卡选择您想使用的模型服务商：
    *   **GLM** (智谱 AI)
    *   **Kimi** (月之暗面)
    *   **Doubao** (字节跳动豆包)
    *   **Custom** (自定义/其他兼容模型)

### 2.1 获取与填写 API Key
*   **填写 Key**：在 **"API Key"** 输入框中粘贴您的 API 密钥。
*   **如果没有 Key？**：点击输入框右侧的 **"Get Key" (获取 Key)** 按钮。程序会自动打开对应服务商的官方控制台网页，请在那里注册账号并申请 API Key。

### 2.2 自定义模型配置 (Custom)
如果您使用的是上述三家之外的模型（或通过中转服务），请选择 **"Custom" (自定义)** 标签：
*   **Model Name (模型名称)**：输入目标模型的 ID，例如 `claude-3-5-sonnet-20241022` 或 `gpt-4o`。
*   **API Key**：输入您的密钥。
*   **API Endpoint (API 端点)**：输入接口地址，例如 `https://api.example.com/v1`。

> **提示**：所有设置在修改后会自动暂存，点击界面右下角的 **"Save Changes" (保存更改)** 按钮可确保持久化保存。

## 3. 激活模型 (Active Model)
配置完成后，您需要告诉程序当前要使用哪个模型。

1.  在界面上方的 **"Active Model" (当前模型)** 区域。
2.  点击您刚刚配置好的模型按钮（如 GLM, Kimi...）。
3.  按钮高亮显示即表示激活成功。此时，系统环境变量已自动同步。

## 4. 设置项目目录 (Project Directory)
指定 Claude Code 运行的工作空间，即您的代码库所在文件夹。

1.  在 **"Launch"** 区域，找到 **"Project Directory" (项目目录)** 一栏。
2.  默认路径为您的用户主目录。
3.  点击右侧的 **"Change" (更改)** 按钮。
4.  在弹出的对话框中选择您的项目文件夹。

## 5. 启动 Claude Code
一切就绪，开始编码！

1.  **Yolo 模式 (可选)**：
    *   勾选 **"Yolo Mode"** 复选框。
    *   此模式会添加 `--dangerously-skip-permissions` 参数，Claude 在执行文件读写或命令执行时将不再逐一询问权限。**请在信任模型输出的情况下谨慎使用。**
2.  点击底部的 **"Launch Claude Code" (启动 Claude Code)** 按钮。
3.  一个新的命令行窗口将弹出，并自动进入 Claude Code 交互界面。

## 6. 其他功能
*   **多语言切换**：点击标题栏上的语言按钮（如 "简体中文"），可在中、英、日、韩、德、法文之间循环切换。
*   **系统托盘**：程序运行时会驻留在系统托盘区。右键点击托盘图标可以：
    *   快速切换模型。
    *   一键启动 Claude Code。
    *   显示/隐藏主窗口。
